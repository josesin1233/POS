===========================================
SISTEMA DE TUTORIALES INTERACTIVOS - NOTAS
===========================================

UBICACION DEL CODIGO:
- Archivo principal: static/js/tutorial-helper.js
- Incluido en: templates/pos/pos_minimalista.html, templates/caja.html, templates/registro.html
- Tambien en: templates/base_production.html, templates/pos/pos.html

COMO FUNCIONA:
1. Boton flotante morado en esquina inferior derecha
2. Draggable - se puede mover a cualquier posicion
3. Guarda posicion en localStorage
4. Al hacer click, muestra modal con PESTANAS para todas las secciones
5. Tutoriales cross-page: puede redirigir a otra pagina y auto-iniciar el tutorial
6. Elementos con fallback: si un selector no existe, intenta un selector alternativo
7. Si un elemento no se encuentra, salta al siguiente paso automaticamente

SISTEMA DE PESTANAS:
- Modal muestra tabs: Punto de Venta, Inventario, Caja, Registro
- Cada tab muestra los tutoriales de esa seccion
- La pagina actual se marca con un punto morado en el tab
- Tutoriales de otra pagina muestran badge "Ir a [Pagina]"

TUTORIALES CROSS-PAGE:
- Si seleccionas un tutorial de otra pagina, se muestra modal de redireccion
- Al confirmar, guarda tutorial pendiente en localStorage (tutorial-pending)
- Al cargar la pagina destino, checkPendingTutorial() lo detecta y auto-inicia
- Timeout de 800ms para esperar que la pagina cargue completamente

ESTRUCTURA DE TUTORIALES:
{
  'nombre-pagina': [
    {
      id: 'id-tutorial',
      title: 'Titulo',
      description: 'Descripcion',
      page: 'nombre-pagina',
      steps: [
        {
          target: '#selector-css',
          message: 'Mensaje',
          position: 'bottom|top|left|right',
          fallback: '#selector-alternativo',  // opcional
          isLast: true|false  // opcional
        }
      ]
    }
  ]
}
